# 30-Day Learning Project Plan: ETL-STOCKS-PIPELINE

## Overview
This document outlines a detailed 30-day plan for the ETL-STOCKS-PIPELINE learning project. The plan is divided into 4 weeks, encompassing daily tasks, learning objectives, deliverables, mini-projects, and success metrics.

---

## Week 1: Understanding ETL Processes

### Day 1: Introduction to ETL
- **Learning Objectives:** Understand the basics of ETL (Extract, Transform, Load) processes.
- **Deliverables:** Write a summary of ETL concepts in a document.
- **Mini-Project:** Research popular ETL tools.
- **Success Metrics:** Completion of the summary and identification of 3 ETL tools.

### Day 2: Data Sources
- **Learning Objectives:** Learn about various data sources.
- **Deliverables:** List and categorize data sources (databases, APIs, files).
- **Mini-Project:** Set up a simple database.
- **Success Metrics:** Completion of the data source list.

### Day 3: Data Extraction
- **Learning Objectives:** Learn how to extract data from different sources.
- **Deliverables:** Code snippets for data extraction.
- **Mini-Project:** Extract data from a chosen API.
- **Success Metrics:** Successful data extraction and storage.

### Day 4: Data Transformation
- **Learning Objectives:** Understand data transformation techniques.
- **Deliverables:** Document transformation techniques.
- **Mini-Project:** Perform data cleaning on the extracted data.
- **Success Metrics:** Cleaned data ready for loading.

### Day 5: Data Loading
- **Learning Objectives:** Learn how to load data into a database.
- **Deliverables:** Code for loading processed data.
- **Mini-Project:** Load data into the database created on Day 2.
- **Success Metrics:** Data is successfully loaded into the database.

### Day 6: Introduction to Python for ETL
- **Learning Objectives:** Learn the basics of Python for data manipulation.
- **Deliverables:** Basic Python scripts for data manipulation.
- **Mini-Project:** Write a script that combines extraction and loading.
- **Success Metrics:** Successful execution of the script.

### Day 7: Review and Reflect
- **Learning Objectives:** Review what has been learned over the week.
- **Deliverables:** Reflection document outlining key learnings.
- **Mini-Project:** Prepare a presentation on Week 1 learnings.
- **Success Metrics:** Completed presentation.
